    Demo

    Demo Page Development Visualization

    November 2015 - January 2016

    My second d3 data visualization. I used a timesheet app to record the data and export a csv to parse with d3. This module is a visualization of the time I spent making this page, the two types of d3 graphs, and the interactive banner.

    Each circle represents a day, where the top of the circle is 12am. Click on a day to veiw a popup of information. Double click to select just one day. This is a single project instance, more modules like this are used to display different school work time data.

    Wyoming

    City of Wyoming - Public Works data entry and reporting tool

    June 2015 - November 2015

    Complete Logs


    A modernization effort for a tool used by the Public Works department of the city of Wyoming. The old tool was a MS Access interface and database, the modernizations became a web app using PHP and MS SQL server as a backend.

    The lifecycle of this project started with a requirments meeting with the Public Works department. After assessing their needs I was left to research what frameworks could be used. When I presented the possible options the IT department managment chose to use PHP, MS SQL server, and whatever front end tools I would need as development progressed. I was the primary developer during my time at Wyoming.

    School Data

    School Data

    2013 - 2015

    Since my 2nd semester at Grand Valley, I have used a timesheet app to record all of my time spent for classes outside of lectures.

    Below are visualizations of that time using d3. The data was imported and processed from csv files. All the images are in svg. There are two sections with non-perfect data capture. Two classes from Spring 2014 were lost due to a failed hard drive and my postponment to backup in the cloud. In Winter 2015 I had a complication installing a custom rom on my phone and lost 2 months worth of a semester.

    Exercise

    Exercise Data

    Jan 2015 - Dec 2015

    Nearly Perfect data capture, for the entire year of 2015, visualized as a stacked bar graph. Each bar represents the number of repetitions done that day. Each exercise is color coded. This was my first attempt at using d3. It is bascially the bar char tutorial, with some slight modifications.
